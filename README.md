# Cat Adoption & Care Guide Web App

## Overview
This web app connects adopters with cats needing homes, and provides guidance on cat care.

## Features
- Cat listings
- Adoption forms
- Care guides
- User accounts

## Tech Stack
- HTML/CSS/JS
- Node.js
- Web API for pet listings
